# Changelog

## 2.0.0
- Added LAN API console for sending raw `cmd`/`data` payloads with optional device/SKU hints.
- Added native color temperature control using the `colorTemInKelvin` field.
- Added scene sender that triggers any official Govee app `sceneId` over LAN.
- Bumped application version to 2.0.0 and documented the expanded LAN API surface.
